# 出题计划

该题目用于OOCL学员摸底调查，目的在于将不同水平的学员分配到不同班级。



## 1. 出题原则

1. 题目范围应当包含HTML/CSS/JS/Java/MySQL。
2. 60%的题目应当是基础题目，难度较低。30%的题目难度中等，10%为高难度题目，用于考察不同学生的水平层次。
3. 题目最好能在1.5h内答题完毕。
4. 建议题目设置为选择题，方便自动打分。



## 2. 题目方案

* **题型与出题范围**

  |   范围   | 简单题 | 中等题 | 难题 |
  | :------: | :----: | :----: | :--: |
  |   HTML   |   5    |   2    |  1   |
  |   CSS    |   5    |   2    |  1   |
  |    JS    |   5    |   2    |  2   |
  |   Java   |   10   |   4    |  2   |
  | Database |   5    |   2    |  1   |
  | mvc/IOC  |   0    |   0    |  1   |

  

## 3. 出题计划

* **Tasking**

  1. 确定出题计划以及出题人员
  2. 出第一版题目
  3. Review第一版题目
  4. 修改第一版题目，产出第二版题目。
  5. 确定最终题目
  6. 交付
